- 设置一个用户：git config --global user.name "Sun"
- 设置用户邮箱：git config --global user.email  "1509258284@qq.com"

- 进入某个盘： cd d:
- 返回上级目录 cd ..

- 查看当前所在目录：pwd

- 查看当前目录有哪些文件：ls
- 
- 创建新文件夹：mkdir 文件夹名
- 删除文件夹： rm -r 文件夹名
- 创建新文件：touch 文件名.后缀
- 删除文件：rm 文件名.后缀



- 在本地仓库打开gitbash，添加一个远程仓库：git remote add 名字  连接地址

- git remote -v 查看是否跟远程仓库连接
